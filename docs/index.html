Emacs vs Vim?

I don't have a strong opinion on it. I can use both. When I use emacs, I prefer using Spacemacs to default emacs.

Spacemacs = Vim Style Command + Emacs Ecosystem. For general purposes except coding, I would use Vim. (I have a dedicated repo for my vim configuration. junkpiano/vim)

On Unix system, especially for cursor navigation, key-binding is closer to Emacs. but for text editting, vim is better. Spacemacs
